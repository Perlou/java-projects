# 企业知识库 - 常见问题

## 产品相关

### Q: 你们的主要产品有哪些？

我们提供以下核心产品：

1. **智能数据平台** - 企业级数据管理解决方案
2. **云原生微服务平台** - 支持容器化部署的微服务框架
3. **AI 开发套件** - 包含大模型 API、向量数据库、Agent 框架
4. **低代码开发平台** - 可视化应用开发工具

### Q: 产品如何定价？

- 基础版：免费，支持 3 个项目
- 专业版：¥999/月，无限项目，优先技术支持
- 企业版：定制报价，私有化部署，专属服务

### Q: 是否支持私有化部署？

是的，企业版支持完整的私有化部署：

- 支持 K8s、Docker 部署
- 提供部署文档和脚本
- 专业团队远程协助

## 技术支持

### Q: 如何获取技术支持？

技术支持渠道：

1. 在线文档：docs.example.com
2. 社区论坛：community.example.com
3. 工单系统：登录后台提交
4. 企业版专属群：7x24 小时响应

### Q: API 调用限制是多少？

- 基础版：100 次/分钟
- 专业版：1000 次/分钟
- 企业版：无限制

### Q: 数据安全如何保障？

我们采用多层安全措施：

- 传输加密：TLS 1.3
- 存储加密：AES-256
- 访问控制：RBAC 权限管理
- 合规认证：ISO 27001、SOC 2

## 账户与计费

### Q: 如何升级账户？

1. 登录账户后台
2. 进入"订阅管理"
3. 选择目标套餐
4. 完成支付

### Q: 支持哪些支付方式？

- 支付宝
- 微信支付
- 银行转账（企业版）
- 对公开票

### Q: 如何申请发票？

登录后台 → 财务中心 → 发票管理 → 申请开票
电子发票即时发送，纸质发票 3-5 个工作日寄出

## 开发相关

### Q: 支持哪些编程语言？

SDK 支持：

- Java（推荐）
- Python
- JavaScript/TypeScript
- Go

### Q: 如何获取 API Key？

1. 注册并登录开发者后台
2. 创建应用
3. 在"API 密钥"页面生成
4. 妥善保管，不要泄露
